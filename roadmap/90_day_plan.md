# 90-Day DSA Plan

## How to use
- Pick the day’s assignment from `problems/daily_assignments/`.
- Implement solutions in `solutions/day_XXX/`.
- Track progress in `roadmap/topic_progress.md`.
- Capture learnings in `notes/`.

## Milestones (suggested)
- Days 1–30: Foundations + core patterns
- Days 31–60: Intermediate (trees/graphs/DP)
- Days 61–90: Mixed sets + timed practice + reviews
